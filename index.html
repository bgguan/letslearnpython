The Print Command

The print command is used to "print" numbers, variables, and words onto the screen.
An example is shown below:

print "Hello"

This would result in the console printing the word: Hello.

Your assignment: On the second print command, type in: My name is (Type in name here) in the parantheses.


Variables

You can assign variables to values by using an equal sign.
Here's an example of a variable.

pizza = "yummy"

You can print variables, too. You cannot have quotation marks, though.

print pizza

As a result, you'll get the word: yummy.

Your assignment: Assign the variables on the editor.
